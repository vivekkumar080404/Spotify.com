# 🎵 Spotify Clone

A responsive Spotify Clone built using **HTML5** and **CSS3**. This project recreates the look and feel of Spotify's web interface with a modern design, responsive layout, and clean user interface.

## 🚀 Features

- 🎧 Spotify-inspired user interface
- 📱 Responsive design
- 🎨 Clean and modern layout
- 📂 Sidebar navigation
- 🎵 Playlist section
- 💻 Built using only HTML and CSS

## 🛠️ Technologies Used

- HTML5
- CSS3
